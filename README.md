# brian d foy's general CLAUDE instructions for Perl modules

Basic stuff I'm refining for Perl modules and the way I like to do things. You
don't have to like all those things, so steal what you like and ignore the rest.

## Installation

Put this in your repo somewhere, and import it into your basic *CLAUDE.md*:

	@CLAUDE-briandfoy-perl.md
